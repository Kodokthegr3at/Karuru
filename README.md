# Karuru 🚀

## 📌 概要
Karuruは、人と人とのコミュニケーションをよりスムーズにすることを目的としたWebアプリケーションです。  
特に、言語や距離の壁を減らし、誰でも簡単に交流できる環境を目指して開発しました。

本プロジェクトは個人開発として、設計から実装までを一貫して行っています。

---

## 🎯 開発目的
- Webアプリケーション開発スキルの向上
- 実践的な開発経験の習得
- 問題解決力の強化
- 将来的に人と人をつなぐサービスの実現

---

## 💡 アプリケーション内容
Karuruは、ユーザー同士が簡単にコミュニケーションできることを目的としたアプリケーションです。

主に以下のような機能を想定・開発しています：
- シンプルで使いやすいユーザーインターフェース
- ユーザー同士の情報共有・コミュニケーション機能
- 今後、リアルタイム翻訳機能の追加を予定

## ▶️ 実行方法（Eclipse + Tomcat 9）

### 必要環境
- Eclipse（Web Developer版推奨）
- Apache Tomcat 9
- Java JDK（8以上）

---

### 手順

1. リポジトリをクローン
```bash
git clone https://github.com/Kodokthegr3at/Karuru.git

Eclipseでインポート

「File」→「Import」

「Existing Projects into Workspace」を選択

クローンしたフォルダを指定

Tomcat 9の設定

「Window」→「Preferences」→「Server」→「Runtime Environments」

「Add」→「Apache Tomcat v9.0」を選択

Tomcatのインストール先を指定

サーバーにプロジェクトを追加

「Servers」タブで右クリック → 「New」→「Server」

Tomcat 9を選択

「Karuru」を追加

実行

サーバーを右クリック → 「Start」

またはプロジェクトを右クリック → 「Run on Server」

🌐 アクセス

ブラウザで以下にアクセス：

http://localhost:8080/Karuru
